
Реализуйте функцию `words_by_letters()`, которая разбирает текст и группирует слова по буквам, отсортированным по алфавиту:

```ruby
sentence = 'hexlet helps people to become developers'
result = words_by_letters(sentence)
# {
#   b: ['become'],
#   d: ['developers'],
#   h: ['helps', 'hexlet'],
#   p: ['people'],
#   t: ['to']
# }
```
