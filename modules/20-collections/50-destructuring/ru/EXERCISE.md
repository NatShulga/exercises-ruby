
Реализуйте функцию `convert()`, которая преобразует переданные на вход данные по примеру:

```ruby
data = [
  ['ruby', 'dynamic', 'strong'],
  ['js', 'dynamic', 'weak'],
  ['c', 'static', 'weak'],
  ['kotlin', 'static', 'strong']
]
# Удаляет вид типизации (второй элемент в массиве)
result = convert(data)
# [
#   ['ruby', 'strong'],
#   ['js', 'weak'],
#   ['c', 'weak'],
#   ['kotlin', 'strong']
# ]
```
